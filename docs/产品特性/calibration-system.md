# MPAC OTA 测试系统 - 智能校准系统

## 功能概述

MPAC OTA 测试系统的校准模块是一套**全自动化、多层级、智能联动**的校准解决方案，专为 5G NR MIMO OTA 测试设计。系统从基础的探头校准到信道级验证，提供完整的校准链路管理，确保测量结果的准确性和可溯源性。

---

## 核心能力

### 1. 多层级校准架构

```
┌─────────────────────────────────────────────────────────────┐
│  Level 5: 系统验证 (TRP/TIS Validation)                      │
├─────────────────────────────────────────────────────────────┤
│  Level 4: 信道校准 (时域/多普勒/空间相关性/静区)               │
├─────────────────────────────────────────────────────────────┤
│  Level 3: 端到端校准 (E2E 补偿矩阵)                          │
├─────────────────────────────────────────────────────────────┤
│  Level 2: RF 链路校准 (LNA/PA/双工器)                        │
├─────────────────────────────────────────────────────────────┤
│  Level 1: 路损校准 (SGH → 探头路径)                          │
├─────────────────────────────────────────────────────────────┤
│  Level 0: 探头校准 (幅度/相位/极化/方向图)                    │
└─────────────────────────────────────────────────────────────┘
```

### 2. 支持的校准类型

| 校准类型 | 功能 | 精度要求 | 有效期 |
|---------|------|---------|--------|
| 探头幅度校准 | TX/RX 增益测量 | ±0.5 dB | 3 个月 |
| 探头相位校准 | 通道间相位一致性 | ±1° | 3 个月 |
| 极化校准 | XPD/轴比测量 | XPD > 20 dB | 6 个月 |
| 路损校准 | SGH 到探头的路径损耗 | ±0.5 dB | 6 个月 |
| RF 链路校准 | LNA/PA 增益校准 | ±0.3 dB | 3 个月 |
| 信道时域校准 | RMS 时延扩展验证 | ±10% | 按场景 |
| 多普勒校准 | 移动场景频偏验证 | ±5% | 按场景 |
| 静区校准 | 幅度/相位均匀性 | ±0.5 dB / ±5° | 6 个月 |

## 技术亮点

### 🔗 智能依赖管理

系统自动追踪校准项之间的依赖关系，当上游校准失效时：
- **级联失效**：自动标记受影响的下游校准项
- **重新校准触发**：智能调度需要更新的校准项
- **依赖可视化**：图形化展示校准项依赖关系

### ⚡ 一键全系统校准

预置多种校准工作流模板，支持一键执行：
- **全系统校准**：按依赖顺序执行完整校准流程
- **频率切换校准**：快速校准新频点
- **日常验证**：快速检查系统状态

### 📊 实时可视化

- **校准时间线**：按时间展示校准历史
- **状态仪表盘**：一目了然的系统健康状态
- **趋势分析**：精度漂移趋势图

### 📋 合规报告生成

支持多种报告格式：
- **ISO 校准证书**：符合 ISO/IEC 17025 标准
- **审计报告**：完整的校准活动记录
- **数据导出**：JSON/CSV 格式导出

### 🚀 相对校准方案 (Quick Mode)

**独创的快速校准模式**，大幅降低日常校准时间：

| 模式 | 测量通道数 | 耗时 |
|------|-----------|------|
| 全量校准 | N 个通道 | ~2 小时 |
| 快速校准 | 1 个参考通道 | ~5 分钟 |

核心机制：
1. **建立基线**：首次全量校准，计算各通道相对参考通道的 Delta 偏移量
2. **快速推导**：后续仅测量参考通道，其他通道值 = 参考值 + Delta
3. **漂移监测**：自动检测漂移，超阈值触发全量校准

## 应用场景

### 场景一：日常测试前校准

测试工程师在开始 OTA 测试前，通过系统仪表盘快速检查校准状态，一键执行"日常验证"工作流，确保测量精度。

### 场景二：频段切换

当测试需求从 n78 频段切换到 n77 频段时，系统自动识别需要更新的路损和相位校准项，执行"频率切换校准"工作流。

### 场景三：年度校准审计

质量管理团队需要审核全年校准活动，系统一键生成审计报告和 ISO 校准证书，满足实验室认证要求。

### 场景四：新暗室部署

新暗室投入使用前，执行"全系统校准"工作流，从探头到信道级完成完整校准链路建立，生成基线报告。
