## 常见问题排查

### 问题1：页面无法加载

**症状**：访问 http://localhost:5173 显示"无法连接"

**解决方案**：
```bash
# 检查服务是否运行
cd /home/user/Meta-3D/gui
npm run dev

# 如果端口被占用，kill进程
lsof -ti:5173 | xargs kill -9
npm run dev
```

---

### 问题2：侧边栏没有"虚拟路测"选项

**可能原因**：
- 代码未保存或未编译

**解决方案**：
```bash
# 停止服务 (Ctrl+C)
# 重新启动
npm run dev
```

---

### 问题3：点击卡片没有反应

**症状**：点击模式卡片，边框不变化

**检查点**：
1. 打开浏览器控制台，查看是否有JavaScript错误
2. 检查React DevTools，确认状态更新
3. 尝试刷新页面（Ctrl+R 或 Cmd+R）

---

### 问题4：图标不显示

**症状**：卡片上的CPU、插头、雷达图标不可见

**可能原因**：
- Tabler Icons未正确导入

**解决方案**：
```bash
# 检查依赖
npm list @tabler/icons-react

# 如果缺失，安装
npm install @tabler/icons-react
```

---

### 问题5：样式显示异常

**症状**：卡片布局错乱，颜色不正确

**可能原因**：
- Mantine CSS未加载

**检查**：
在浏览器中查看 Network 标签，确认以下资源加载：
- `@mantine/core/styles.css`

**解决方案**：
检查 `main.tsx` 是否正确导入：
```typescript
import '@mantine/core/styles.css'
```
